# TMUX
A seguir as configurações do `tmux`.

## Setup
Basta instalar o `tmux` via terminal com o seguinte comando:

~~~bash
sudo apt install tmux
~~~

## Estrutura de arquivos
### tmux.conf
Arquivo que faz as configurações do `tmux`. Não são muitas configurações,
portanto, utilizamos apenas um arquivo para todas as configurações
